# MergeSort 

## Challenge 

Implement a merge sort algorithm where sorted items are broken into smaller collections and compared with the item next to them.

## What is it used for?

```
- Stable sorting
- Can be used to sort huge amounts of data.
- Worst case O(nlog(n))
```
